# 🚀 RencanaKita

### Smart Investment Recommendation Platform

> Helping users make smarter investment decisions based on risk profile, financial goals, and investment horizon.

---

## 💡 What is RencanaKita?

**RencanaKita** is an intelligent investment recommendation system designed to simplify how users choose investment products.

Instead of guessing between deposits, mutual funds, bonds, or stocks — users receive **data-driven, personalized recommendations** based on their financial profile.

We turn complex investment decisions into **clear, guided choices**.

## 🎯 Problem We Solve

Most beginner investors struggle with:

- ❌ Not understanding risk tolerance
- ❌ Choosing the wrong investment products
- ❌ Panic selling during market volatility
- ❌ Lack of structured financial planning

👉 Result: poor investment decisions and missed financial goals.

---

## ✅ Our Solution

RencanaKita provides:

- 🧠 Automated risk profiling
- 🎯 Personalized investment recommendations
- 📊 Data-driven decision support
- 📈 Long-term financial alignment

---

## ⚙️ How It Works

1. User inputs financial profile  
   (age, income, risk tolerance, investment horizon, goals)

2. System analyzes user behavior & profile  
   using classification + rule-based logic

3. Recommendation engine matches user with suitable products  
   (e.g., Deposits, SBN, Mutual Funds, Stocks)

4. User receives personalized investment suggestions
